# Bleve project

This is my university assignment for COMP6013: Explainable Approaches to Machine Learning at Curtin University.

I completed this project on my own as part of the unit requirement.

## What the assignment was about?
The task was to predict peak pressure in a BLEVE environment using machine learning.

## What I did
- Cleaned and preprocessed the data
- Created new features
- Tested three models:
  - Random Forest
  - XGBoost
  - CatBoost
- Compared model performance using MAPE and R²
- Used SHAP for model interpretation
- Generated final predictions for the test set

## Final model
CatBoost was the best model based on my validation results.
